# figma-mcp-relay skills

Claude Code skills that orchestrate figma-mcp-relay tools.

## Install

Copy each subdirectory into `~/.claude/skills/`:

```bash
cp -r packages/skills/figma-codegen ~/.claude/skills/
cp -r packages/skills/figma-sync-tokens ~/.claude/skills/
cp -r packages/skills/figma-audit ~/.claude/skills/
```

> Skills are intentionally **not** published to npm or a third-party registry — they live in this repo and are copied by the user.
